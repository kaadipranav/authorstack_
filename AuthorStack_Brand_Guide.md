# AuthorStack -- Complete Brand & Product Guide

## 1. Overview

AuthorStack is a multi-phase indie author platform combining analytics,
AI assistance, community, and future distribution tools. It aims to give
authors a professional dashboard that unifies sales, audience insights,
and writing-support tools.

## 2. Core Idea

A platform that turns scattered indie author data into a clear story: -
Sales analytics - Reader behavior insights - AI publishing assistant -
Community tools - Future distribution channels

Tagline: **Data That Tells Your Story.**

## 3. Brand Identity

### 3.1 Logo

-   Monochrome palette (black/white/grey)
-   Editorial layout, clean spacing
-   Modern typefaces: Geist / Inter / Satoshi

## 4. Tech Stack

-   **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind
-   **Backend:** Supabase (Auth, DB)
-   **AI:** OpenAI / future LLM integrations
-   **Styling:** shadcn/ui, Tailwind
-   **State:** React server components + client islands
-   **Integrations:** Gumroad (first), later KDP, Draft2Digital, Kobo,
    StoryGraph API, etc.

## 5. Project Structure

-   `/app` -- Routes, pages
-   `/components` -- UI components
-   `/features` -- Feature modules
-   `/lib` -- Utils, configs, client/server helpers
-   `/supabase` -- Types & schema
-   `/public` -- Assets
-   Root configs: package.json, tsconfig, postcss, next.config

## 6. Execution Roadmap (5 Phases)

### **Phase 1 --- MVP Dashboard (Current \~60% complete)**

-   Supabase auth
-   Profiles
-   Base UI
-   Sales dashboard skeleton
-   Gumroad integration (upcoming)
-   Book & sales models

### **Phase 2 --- Author Tools + Community**

-   Leaderboard (Book discovery ranking)
-   Author profiles with metrics
-   Lightweight community feed
-   Book uploads
-   Early AI suggestions

### **Phase 3 --- AI Publishing Assistant (The mascot)**

-   AI that analyzes book performance
-   AI blurbs, descriptions, launch plans
-   AI that critiques chapters
-   Readability + sentiment scoring
-   Predictive sales analytics
-   Personalized advice

### **Phase 4 --- Distribution & Automation**

-   Multi-platform importer (Gumroad, KDP, Draft2Digital)
-   Automated launch scheduling
-   Automated email campaigns
-   Reader segmentation insights

### **Phase 5 --- Ecosystem & Marketplace**

-   AuthorStack print-on-demand
-   Marketplace for editors, designers, beta readers
-   AuthorStack Publishing imprint
-   Mobile app
